# SmartCookBook Media

This folder contains screenshots, mockups, and demo materials for SmartCookBook.

## Folder Structure

```
media/
├── screenshots/          # Application screenshots
├── mockups/             # UI design mockups
├── diagrams/            # Architecture diagrams
├── videos/              # Demo recordings
├── presentations/       # Demo scripts
└── SmartCookBook_Showcase.html   # Interactive showcase page
```

## Screenshots

Screenshots are stored in `screenshots/` folder. Some screenshots were captured during development showing various features.

Open `SmartCookBook_Showcase.html` in a browser to view the interactive showcase page.
